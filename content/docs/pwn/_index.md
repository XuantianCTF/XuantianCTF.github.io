PWN 方向研究二进制程序的漏洞发现与利用技术，是 CTF 中技术含量最高的方向之一。掌握 PWN 需要深入理解操作系统、编译原理和计算机体系结构。

### 什么是 PWN？

PWN 源自网络用语，意为"攻破"或"控制"。在 CTF 中，PWN 指的是利用二进制程序中的漏洞来获取控制权或执行任意代码。

PWN 题目通常提供一个有漏洞的程序，你需要编写 exploit 来利用漏洞，最终获取 shell 或读取 flag 文件。

### 为什么学习 PWN？

1. **深入理解系统**：PWN 让你真正理解计算机如何执行代码
2. **漏洞挖掘**：掌握 PWN 技能可以发现高级漏洞
3. **攻防兼备**：了解攻击才能更好地防御
4. **薪资天花板高**：二进制安全专家是安全行业最稀缺的人才

### 核心知识点

| 知识领域 | 内容 |
|----------|------|
| 内存管理 | 栈、堆、虚拟内存、页表 |
| 漏洞类型 | 栈溢出、堆溢出、UAF、格式化字符串 |
| 利用技术 | ROP、JOP、堆利用、Shellcode |
| 安全机制 | NX、ASLR、PIE、Canary、CFI |
| 调试工具 | GDB、pwndbg、pwntools |

### 学习路线

1. 学习 C 语言和汇编语言
2. 理解计算机内存管理（栈、堆、函数调用）
3. 掌握常见漏洞利用技术（栈溢出、堆利用、ROP）
4. 了解安全防护机制及其绕过方法
5. 使用 pwntools 编写自动化利用脚本
