## 流量分析

流量分析是 CTF 中常见的杂项题型，需要从网络抓包文件中提取信息。

### 常见协议

- HTTP/HTTPS
- FTP
- DNS
- ICMP
- TCP/UDP

### 常见操作

- 追踪 TCP 流
- 导出 HTTP 对象
- 过滤特定协议
- 提取传输的文件

### 工具

- **Wireshark**：图形化流量分析
- **tshark**：命令行流量分析
- **NetworkMiner**：网络取证
