密码学是研究信息加密与解密的学科，是 CTF 中极具挑战性的方向。从古典密码到现代密码体系，密码学融合了数学、计算机科学和信息论等多学科知识。

### 什么是密码学？

密码学（Cryptography）是研究如何保护信息安全的技术科学。它涉及加密、解密、数字签名、哈希函数等多个方面。

在 CTF 中，密码学题目通常要求你破解某种加密算法或找出密码系统的弱点。这需要你理解数学原理，而不仅仅是会使用工具。

### 为什么学习密码学？

1. **数学之美**：密码学是数学在实际应用中的完美体现
2. **安全基石**：现代互联网安全（HTTPS、数字证书）都依赖密码学
3. **智力挑战**：密码学题目通常需要创造性思维
4. **研究价值**：量子计算时代，后密码学是热门研究方向

### 核心知识点

| 知识领域 | 内容 |
|----------|------|
| 编码方式 | Base64、Hex、URL编码、摩斯电码 |
| 古典密码 | 凯撒、维吉尼亚、栅栏、培根 |
| 现代密码 | AES、DES、RSA、ECC |
| 哈希函数 | MD5、SHA1、SHA256、HMAC |
| 数论基础 | 模运算、欧拉定理、中国剩余定理 |

### 学习路线

1. 掌握各种编码方式（Base64、Hex、摩斯电码）
2. 学习古典密码原理与破解方法
3. 理解现代密码学基础（对称/非对称加密、哈希函数）
4. 深入研究 RSA 算法及其常见攻击
5. 在 CryptoHack 等平台持续练习
