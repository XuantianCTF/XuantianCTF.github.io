Web 安全是 CTF 比赛中最常见也是最适合新手入门的方向。由于 Web 应用无处不在，掌握 Web 安全技术对于理解整个网络安全生态至关重要。

本方向涵盖 OWASP Top 10 中的各类漏洞，包括注入、XSS、文件上传、反序列化等。我们将从基础原理讲起，结合实际案例和靶场练习，帮助你建立完整的 Web 安全知识体系。

### 学习路线

1. 掌握 HTTP 协议与浏览器工作原理
2. 学习 PHP/Python 等后端语言基础
3. 理解常见 Web 漏洞原理
4. 使用 Burp Suite 等工具进行实战
5. 在靶场中反复练习，积累经验
